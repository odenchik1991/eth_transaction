# Send a Simple Ethereum Transaction

This script sends a basic ETH transaction using the Goerli Testnet. 

**WARNING:** Only use testnet accounts for safety.

## Requirements

- Python 3.x
- web3.py library
- Infura Project ID
- Test wallet with ETH (Goerli/Sepolia)
